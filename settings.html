Edit game speed:

Note: This game was designed to be fast was initially set to run at maximum speed (1). This worked fine for a PII 450 or therabouts back in the day, but on faster computers I soon learned it was too much! A PIII 850 required a value of about 25. In short - the higher the value, the slower the game, I've set the default to 30. You can set it as fast or slow as you like!
